# Partner Setup

## Required Software
- Node.js LTS (recommended: v20 or v22)
- npm (included with Node.js)

## Verify Installation
```bash
node -v
npm -v
```

## Run the Project
1. Unzip the project folder.
2. Open terminal in the `youth-empowerment` folder.
3. Install dependencies:
```bash
npm install
```
4. Start development server:
```bash
npm run dev
```
5. Open in browser:
- http://localhost:3000
- If 3000 is busy, use the port shown in terminal (for example 3001).

## Production Mode (Optional)
```bash
npm run build
npm run start
```
